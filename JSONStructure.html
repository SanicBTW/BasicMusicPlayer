The settings json structure should be like this:

{
    "categories": ["TimeBar", "TimeDisplay", "Window"],
    "configuration":
    {
        "TimeBar":
        {
            "backgroundColor": [number limit 255, number limit 255, number limit 255],
            "Color": [number limit 255, number limit 255, number limit 255],
            "Width": number percentage limit is obvious,
            "Height": number px no limit i guess,
            "display time left instead of cur time": boolean (true, false)
        },
        "TimeDisplay":
        {
            "display time left instead of cur time": boolean (true, false)
        },
        "Window":
        {
            "update window title": boolean (true, false),
            "display song name": boolean (true, false),
            "display time left": boolean (true, false)
        }
    }
}

You can copy it and modify the values to match your likings, types are written in the values

The file name doesnt matter, as long as the file extension is .json and the structure is like that then everything is fine
